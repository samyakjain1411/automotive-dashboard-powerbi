# 🚗 Automotive Sales Performance & Inventory Dashboard | Power BI

## 📌 Project Overview
Developed an end-to-end Business Intelligence solution to monitor and optimize sales performance and dealership operations in the automotive sector. The project transforms raw transactional data into meaningful insights for better decision-making.

---

## 🎯 Business Objective
- Track sales performance across regions and time periods  
- Analyze dealership efficiency  
- Optimize inventory management  
- Identify high and low performing vehicle segments  

---

## 📊 Key Features
- 📈 Year-over-Year (YoY) Growth Analysis  
- 💰 Total Sales & Revenue Tracking  
- 📦 Inventory Turnover Analysis  
- 🌍 Region-wise Sales Insights  
- 🚗 Vehicle Category Performance  

---

## 🧠 Data Modeling
- Built a structured data model connecting:
  - Sales Data  
  - Inventory Data  
  - Dealership Data  
  - Regional Data  
- Implemented relationships for efficient filtering and analysis  

---

## ⚙️ DAX Measures Used
- Total Sales  
- Revenue Growth  
- Inventory Turnover  
- YoY Growth %  

---

## 🎨 Dashboard Design
- Interactive and user-friendly interface  
- Dynamic filters (Region, Category, Time)  
- Drill-down capabilities for detailed analysis  

---

## 🔍 Key Insights
- Identified slow-moving inventory  
- Highlighted high-performing vehicle segments  
- Discovered regional variations in sales performance  
- Enabled better inventory and sales strategy decisions  

---

## 🛠 Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Microsoft Excel
